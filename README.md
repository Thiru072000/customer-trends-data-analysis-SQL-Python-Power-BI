👨🏻‍💻 Customer Behavior Analytics | SQL • Python • Power BI

This project demonstrates a complete end-to-end Data Analytics workflow designed to simulate real-world business analysis scenarios. It showcases how raw customer transaction data can be transformed into meaningful insights using Python, SQL, and Power BI.

The project reflects practical responsibilities of a Data Analyst, including data cleaning, querying, KPI development, and dashboard creation.

📌 Project Objective

The objective of this project is to analyze customer shopping behavior and uncover actionable insights that can help businesses improve revenue, customer retention, and product performance.

This project covers:

Data cleaning and preparation using Python

Business analysis using SQL

Interactive dashboard creation using Power BI

KPI development and insight reporting

🛠️ Tools & Technologies Used

Python (Pandas, NumPy) – Data cleaning and exploratory analysis

SQL (MySQL / PostgreSQL) – Business query analysis

Power BI – Dashboard and visualization

Jupyter Notebook – Analysis workflow

GitHub – Version control and documentation

📊 Project Workflow
1️⃣ Data Preparation & Cleaning (Python)

Loaded raw customer dataset

Handled missing values

Standardized column names

Performed category-wise imputations

Conducted exploratory data analysis

2️⃣ Business Analysis (SQL)

Created structured database

Loaded dataset into SQL

Wrote business-driven queries to answer key questions:

Which customer segment generates highest revenue?

What is the average purchase value?

Which product category performs best?

What are monthly revenue trends?

How does customer rating vary by category?

3️⃣ Dashboard & Visualization (Power BI)

Built an interactive dashboard including:

Total Revenue KPI

Average Purchase Value

Customer Segmentation

Category Performance

Trend Analysis

Rating Distribution

The dashboard enables business stakeholders to quickly identify growth opportunities and performance gaps.

📈 Key Insights Generated

Revenue concentration among specific customer segments

High-performing product categories

Purchase trends across time periods

Behavioral patterns influencing spending

Category-wise rating distribution analysis
